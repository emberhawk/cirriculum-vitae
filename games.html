These are games I haved created in the last year, most of them are guided tutorials or code alongs, I present them to you for a more clear understanding.

Just a heads up these games were mainly made for the pc and won't really work on the mobile devices, either due to scale or input. I may update in the future for now just use a keyboard and mouse.

Minecraft Clone

This minecraft clone I made with the voxel course. It uses fractual browning motion to procedural generate a world. This clone is effectivity in creative mode. There is performance issues but if you move to the edge of the map it will generate a new part of the world. This was created to better understand how minecraft blocking/chunk system and how it works.
